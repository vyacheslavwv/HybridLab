## 🎓 ВЕРОЯТНЫЕ ВОПРОСЫ ЖЮРИ

**Q: Как вы реализовали интерактивность на Canvas?**  
**A:** Используя `onMouseMove` и интерполяцию значений из массива данных.

**Q: Почему glassmorphism?**  
**A:** Это современный тренд в UI дизайне, создаёт эффект глубины и профессионализма.

**Q: Почему Signal Registry?**  
**A:** Паттерн делает код масштабируемым. Добавить новый сигнал — просто добавить объект в реестр.
