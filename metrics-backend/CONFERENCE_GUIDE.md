## 🎮 Как использовать новые функции

### 1. **Запуск симуляции**
```
1. Откройте index.html в браузере
2. Нажмите ▶ СТАРТ в панели управления
3. Двигайте ползунок ДРОССЕЛЬ влево-вправо для изменения нагрузки
```

### 2. **Использование осциллографа**
```
1. Перетащите серый круг (щуп) на одну из тестовых точек:
   - БАТАРЕЯ (синий) — 144В
   - МОТОР (зелёный) — 120В
   - ДВС (оранжевый) — 12В

2. На осциллографе появится волна сигнала

3. Наведите мышку на график, чтобы увидеть точное значение в подсказке

4. Переключайтесь между инструментами кнопками "~ Осциллограф" и "V Мультиметр"
```

### 3. **Экспорт данных в CSV**
```
1. Запустите симуляцию и дождитесь накопления данных (~1-2 сек)

2. Нажмите кнопку  ЭКСПОРТ

3. Файл гибридной-sim-export-ДАТА.csv скачается на ваш компьютер

4. Откройте в Excel или Google Sheets для анализа
```

### 4. **Debug Mode (горячая клавиша D)**
```
1. Нажмите клавишу D (или Shift+D)

2. В нижнем правом углу появится панель с:
   - Текущий FPS
   - Все параметры симуляции в реальном времени
   - Состояние подключённого щупа

3. Нажмите D ещё раз, чтобы скрыть
```

## 🎯 Что спросит кодер и как ответить

### ❓ «Почему вы используете Signal Registry вместо простого switch?»

**Ответ:**
> Signal Registry — это паттерн, который делает код масштабируемым. Вместо добавления новых if/else, вы просто добавляете новый объект в реестр. Это также упрощает тестирование и поддержку.

### ❓ «Зачем вам FPS в состоянии компонента?»

**Ответ:**
> FPS помогает мониторить производительность приложения. Это полезно для отладки и верификации того, что симуляция работает гладко. Debug Panel показывает это значение.

### ❓ «Как вы реализовали интерактивность на Canvas?»

**Ответ:**
> Я добавил `onMouseMove` и `onMouseLeave` обработчики, которые отслеживают позицию курсора. Затем на основе позиции интерполирую значение из массива данных и отрисовываю подсказку с помощью контекста Canvas.

### ❓ «Glassmorphism выглядит красиво, но есть ли браузер, где это не работает?»

**Ответ:**
> Glassmorphism использует `backdrop-filter`, который поддерживается во всех современных браузерах. Для старых браузеров я добавил полупрозрачный фон как fallback через `background: rgba()`.

### ❓ «Почему вы сохраняете историю в состояние?**

**Ответ:**
> История нужна для экспорта CSV. Я ограничиваю её 100 последних точками, чтобы не нагружать память. Это достаточно для анализа в Excel без лага.
