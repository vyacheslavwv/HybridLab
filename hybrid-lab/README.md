# Hybrid Lab

Интерактивная образовательная платформа для изучения гибридных электрических автомобилей.  
Школьный проект по теме «Электрификация транспорта».

**Стек:** React 19 · TypeScript · Vite 7 · Zustand 5 · Tailwind CSS 4 · Canvas API

---

## Что умеет платформа

### Главная страница — Dashboard

- **Живая телеметрия** в шапке: SOC батареи, обороты электромотора и ДВС обновляются в реальном времени пока симуляция запущена
- **Прогресс курса** — дуговой индикатор показывает сколько из 5 модулей пройдено; данные сохраняются в `localStorage`
- **Bento-сетка** с быстрым доступом ко всем разделам: карточки симулятора, калькулятора, топологии, глоссария
- Индикатор статуса симуляции `SIM · RUNNING / IDLE` в нижней части бокового меню

---

### Симулятор гибридного привода

Физическая модель HEV в реальном времени с частотой 60 FPS.

**Панель управления:**
- Слайдер дросселя (0–100 %)
- Кнопки **Старт / Стоп** симуляции
- Переключение режимов работы привода

**Четыре режима гибрида:**

| Режим | Как работает |
|-------|-------------|
| **EV** (электро) | Только электромотор. ДВС выключен, батарея разряжается |
| **Parallel** (параллельный) | ДВС и мотор работают вместе — максимальная тяга |
| **Series** (последовательный) | ДВС вращает генератор → заряжает АКБ → мотор крутит колёса |
| **Charging** (зарядка) | ДВС заряжает батарею, автомобиль стоит |

**Приборы:**
- **4 KPI-карточки** с дуговыми gauge: SOC батареи, обороты мотора, обороты ДВС, положение дросселя
- **Осциллограф** (Canvas, HiDPI) — отображает сигнал с подключённой точки; буфер 512 сэмплов
- **Цифровой мультиметр** — показывает напряжение в точке подключения, метку сигнала и единицу измерения
- **SVG-схема привода** — анимированные потоки энергии между узлами
- **Перетаскиваемый щуп** — наведите и зафиксируйте на одной из четырёх точек: `BATT · MOTOR · ICE · GEN`; осциллограф и мультиметр переключаются автоматически

### Калькулятор TCO

Сравнение полной стоимости владения за выбранный срок для трёх типов автомобилей.

**Параметры (слайдеры):**
- Годовой пробег (5 000 — 80 000 км/год)
- Цена топлива (40 — 120 руб/л)
- Цена электричества (3 — 20 руб/кВт·ч)
- Срок владения (1 — 15 лет)

**Результаты:**
- Столбчатый **Canvas-график** сравнения ДВС / Гибрид / Электро
- Строка «выгоднее» у наиболее экономичного варианта
- Расшифровка по статьям: покупка, топливо/электричество, техобслуживание

### Топология схемы

Практическое задание: собери правильную схему **последовательного гибридного привода**.

- **Drag-and-drop** — перетащи компоненты из палитры в 6 слотов
- Компоненты: ДВС · Генератор · Батарея · Инвертор · Электромотор · Колёса
- **Мгновенная проверка** — слоты подсвечиваются зелёным (верно) или красным (ошибка)
- **Инженерные подсказки** — при неправильном порядке объясняют почему так нельзя
- **Прогресс-бар** заполненности (0/6 → 6/6)
- Кнопка **Сбросить**

### OBD-II Диагностика

Симуляция подключения к бортовой диагностике автомобиля.

**Выбор адаптера:**
- ELM327 USB (COM3)
- ELM327 Bluetooth (BT-OBD2)
- J2534 PassThru (COM4)

**Процесс подключения** (имитация AT-команд ELM327):
```
AT Z  →  сброс адаптера
AT SP 0  →  автоопределение протокола  →  ISO 15765-4 (CAN 500 kbps)
0100  →  поддерживаемые PID
Подключено к ECU
```

**После подключения (живые данные, обновление каждые 400 мс):**
- Скорость автомобиля, обороты ДВС — из симулятора
- SOC HV батареи, напряжение HV шины — из симулятора
- Момент электромотора — из симулятора
- Температура масла — случайное значение ±2 °C

**Коды неисправностей (DTC):** 5 демо-кодов с уровнями критичности; кнопки «Сброс» на каждый код и «Сбросить все» (доступны только при подключённом адаптере).

**Журнал событий** — список AT-команд и ответов с временными метками.

**Раскрывающийся блок** с объяснением что такое OBD-II / ELM327 / J2534 — для понимания без специальных знаний.

### Глоссарий

15 терминов гибридных систем с быстрым поиском.

- Поиск по аббревиатуре, русскому названию и полному английскому наименованию
- Каждый термин — цветной бейдж в своей акцентной цветовой схеме
- Три колонки: аббревиатура · расшифровка · определение
- Счётчик совпадений

Термины: SOC · HEV · ICE · PHEV · BEV · Regen · OBD · DTC · CAN · ECU · HV · IPM · IGBT · MG1/MG2 · PSD

## Запуск

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # сборка в dist/
npm run preview   # предпросмотр сборки
npm run lint      # ESLint
```

Требования: **Node.js 18+**, **npm 9+**

## Структура проекта

```
src/
├── App.tsx                      # Роутинг, боковое меню, layout
├── index.css                    # Дизайн-система (CSS-переменные, утилиты)
├── domain/                      # Чистая логика — без React, без UI
│   ├── physics/SimulationEngine.ts
│   ├── physics/SignalRegistry.ts
│   └── strategies/              # EVMode, ParallelHybridMode, SeriesHybridMode, ChargingMode
├── application/
│   └── stores/simulationStore.ts  # Zustand: состояние + game loop
├── presentation/
│   ├── components/              # Oscilloscope, ControlPanel, CircuitSchematic, DraggableProbe
│   ├── instruments/             # MultimeterDisplay, tcoChart.ts (Canvas)
│   └── pages/                   # TCOCalculator, TopologyBuilder, DiagnosticsPage, GlossaryPage, LiteraturePage
└── infrastructure/config/       # Реестры модулей и компонентов
```

Подробное описание архитектуры — в [ARCHITECTURE.md](./ARCHITECTURE.md).

## Дизайн

Тёмная тема на CSS-переменных, glassmorphism-карточки.  
Шрифт для значений приборов: **JetBrains Mono**.  
Основные акценты: Indigo (навигация) · Emerald (активно/успех) · Amber (ДВС/предупреждение) · Cyan (электро) · Rose (ошибка).
